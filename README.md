# PPL - Home Assignment 1 (Spring 2026)

This repository contains my solution for the first assignment in the **Principles of Programming Languages (PPL)** course at Ben-Gurion University of the Negev.

## Project Overview
The assignment focuses on the transition from imperative to functional programming, using TypeScript and L3 (a subset of Scheme).

### Content:
* **Part 1: Theoretical Questions** - Exploration of programming paradigms and type analysis.
* **Part 2: Functional Programming in TypeScript** - Implementing functions using pure functional constructs and the Ramda library.
* **Part 3: Introduction to Monads** - Working with the `Result<T>` monad for safe error handling.
* **Part 4: Programming in L3** - Implementing recursive procedures and list manipulations.
